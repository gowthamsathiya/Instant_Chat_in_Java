Pre-requisite:

Java 1.5+
Eclipse
Port 80 to be free to run server

How to import project into eclipse?

Open eclipse.
Goto file->Import->Import existing project into workspace->Browse
Select both goChat-Server and goChat-Client and click finish

How to run?

Select goChatServer.java in goChatServer application and hit run or Ctrl+F11 
Select goChatClient.java in goChatClient application to open the client interface and start chatting
